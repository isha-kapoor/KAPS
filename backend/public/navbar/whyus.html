If the waste is not properly disposed it releases Methane and if it is burnt it releases Carbon dioxide. Both the gases are harmful, therefore as a Farmer if you join us you will become the helping hand to save the environment.
The farmers should not worry about how they will dispose the waste and the private companies should not worry about the raw materials they need to make the products as the collection centre performs that job.
If you do not want the waste that comes from farmers to go unused, as the private companies working in this domain you can join hands and use this waste as raw materials to make useful products.
